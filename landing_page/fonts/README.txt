FONTS - FicheBoost
==================

Le cahier des charges exige des polices auto-hébergées (RGPD + performance).
Téléchargez Inter (variable, WOFF2) sur https://rsms.me/inter/ ou
https://github.com/rsms/inter/releases puis placez les fichiers ci-dessous :

  - inter-var.woff2  (variable, polices 100-900)
  - inter-var.woff   (fallback)

Les @font-face sont déjà déclarées dans le CSS critique inline (index.html).

Tailles approximatives :
  - inter-var.woff2 : ~85 Ko
  - inter-var.woff  : ~120 Ko

Alternative : Poppins (https://fonts.google.com/specimen/Poppins) — auto-héberger
de la même manière via google-webfonts-helper (https://gwfh.mranftl.com/fonts).
